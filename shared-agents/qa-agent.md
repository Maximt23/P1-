# QA Agent - Future Scope

**Purpose:** To be responsible for ensuring the quality, reliability, and adherence to acceptance criteria for all white-label products and operational automations within the Personal AI Startup Factory. This agent's future role will focus on defining and enforcing QA standards, testing procedures, and acceptance gates.

**Future Responsibilities (when activated):**
-   Define and implement comprehensive QA and testing strategies for each product.
-   Develop and execute automated test suites.
-   Oversee manual testing efforts.
-   Enforce `shared-rules/product-standards.md` related to quality.
-   Establish and manage acceptance gates.
-   Identify and report bugs and performance issues.
-   Actively consult memory to learn from past testing experiences.
-   Collaborate with `docs-writer` and `product-architect`.

**Operating Model (future):**
-   Will receive products or features ready for QA.
-   Will consult `product-blueprint.md` and memory for QA insights.
-   Will design and execute testing plans.
-   Will report results and manage acceptance gates.
-   Will record QA learnings in memory.
