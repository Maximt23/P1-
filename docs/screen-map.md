# Screen Map (MVP)

This document lists the primary screens required for the WearWise MVP.

## Core Navigation Screens
1.  **OnboardingScreen**: Initial app introduction, permission requests, research consent.
2.  **WardrobeLibraryScreen**: Displays all user-added clothing items.
3.  **OutfitSuggestionsScreen**: Displays the daily ranked outfit suggestions.
4.  **QuickColorCheckCameraScreen**: Dedicated live camera view for color identification.
5.  **SettingsScreen**: User preferences, accessibility options, permission management.

## Secondary / Flow-Specific Screens
6.  **WeeklyRoutineSetupScreen**: Allows users to set up their weekly outfit routines.
7.  **AddClothingItemScreen**: Interface for adding new clothing items (photo or manual).
8.  **ItemDetailEditScreen**: Displays and allows editing of a single clothing item's details.
9.  **ManualOutfitBuilderScreen**: Interface for users to construct outfits from their wardrobe.
10. **SavedOutfitsScreen**: Displays outfits previously saved by the user.
11. **OutfitHistoryScreen**: Shows a chronological log of worn outfits.
