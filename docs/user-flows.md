# User Flows (High-Level)

This document outlines high-level user flows for key MVP features.

## 1. Onboarding & Setup
*   **Flow**: User downloads app -> First launch -> Onboarding screens (welcome, permissions intro, research consent intro) -> Initial wardrobe setup prompt.

## 2. Adding a Clothing Item
*   **Flow**: User navigates to Wardrobe -> Tap "Add Item" -> Choose Photo or Manual Entry -> Capture/Select Photo (if photo) -> AI processes / User manually inputs details (type, color, tuck guidance) -> Edit labels -> Save item.

## 3. Getting Outfit Suggestions
*   **Flow**: User opens app (lands on Suggestions screen) -> Views 3 ranked suggestions -> Can refresh manually -> Can log outfit as worn / save / give feedback.

## 4. Quick Color Check
*   **Flow**: User taps "Quick Color Check" icon -> Live camera view opens -> Points camera at clothing -> App displays likely color label, confidence, and swatch -> User can freeze frame, continue, or save to wardrobe.

## 5. Viewing Wardrobe & Editing Items
*   **Flow**: User navigates to Wardrobe Library -> Views list of items -> Taps an item -> Views Item Detail/Edit Screen -> Can edit labels, colors, tuck guidance, or archive/delete item.

## 6. Managing Settings
*   **Flow**: User navigates to Settings -> Accesses preferences for accessibility, permissions (research, location), and app behavior.
