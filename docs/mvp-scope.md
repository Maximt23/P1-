# MVP Scope

The Minimum Viable Product (MVP) for WearWise includes the following core features:

*   **Wardrobe Library**: A central place to manage all clothing items.
*   **Item Entry**: Ability to add clothing items via photo or manual input.
*   **Editable Clothing Labels**: Users can customize labels for their clothes.
*   **Outfit Suggestions**: Automated suggestions for daily outfits.
*   **Manual Outfit Builder**: Users can create outfits manually.
*   **Outfit History**: A log of previously worn outfits.
*   **Saved Outfits**: Users can save favorite outfit combinations.
*   **Weekly Routine Setup**: Configure outfits for a weekly schedule.
*   **Optional Weather-Aware Ranking**: Outfit suggestions can factor in local weather if permission is granted.
*   **Optional Research Consent**: Users can opt-in for research data sharing without impacting core app functionality.
*   **Settings and Accessibility Preferences**: Customizable app settings, including accessibility features.
*   **Quick Color Check Camera**: A dedicated feature to identify likely colors of clothing in real-time.
